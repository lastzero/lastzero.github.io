Howto: Dunkle Gnome Themes und Firefox/Thunderbird

Michael Mayer
Sep 2, 2006

Eigentlich ist es ganz einfach ein dunkles Gnome-Theme zu verwenden: Downloaden und per Drag-and-Drop installieren. Nur wird man, uebrigens auch unter OS X, alsbald feststellen, dass man die Menues von Firefox und Thunderbird 1.5 nicht mehr lesen kann, weil der Font dort unveraendert schwarz ist. Das macht sich auf einem schwarzen Hintergrund nicht wirklich gut. Es gibt jetzt 3 Moeglichkeiten:

  1. Resigniert das schoene neue Theme deinstallieren. Nicht Sinn der Sache.
  2. Ein Firebird/Thunderbird Theme installieren, dass die Farben entsprechend anpasst. Will man eigentlich auch nicht, weil man damit dann auch 1000 andere Design-Aenderungen hat, die nicht zum Theme passen.
  3. Man aendert die userChrome.css :)

Ich habe keine Ahnung wie der Durchschnitts-User auf 3. kommen soll, aber deshalb schreibe ich das hier. Die userChrome.css muss man natuerlich erst anlegen, wobei man dazu userChrome-example.css als Vorlage verwenden kann. Die Datei liegt ueberlicherweise in

/home/[username]/.mozilla/firefox/[profile]/chrome/

Um den Menu-Font weiss statt schwarz anzuzeigen muss man nur die folgenden Zeilen einfuegen:

menu {
   color: #fff !important;
   }

Unter OS X wird es entsprechend funktionieren. Windows-User haben keine dunklen Themes zur Auswahl – damit stellt sich das Problem nicht.
