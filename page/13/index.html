Zend Framework: Unicode DB adapter for SQL Server

May 31, 2007

Yeah, I finally got it managed to publish my proposal for the Zend Framwork ODBTP database adapter component: http://framework.zend.com/wiki/display/ZFPROP/Zend_Db_Adapter_Odbtp_Mssql The default MS SQL database driver does not work with Unicode, as SQL Server expects Unicode strings in the UCS-2 encoding (UCS-2 contains null bytes and the driver is not binary-safe). plat_forms photos online

May 29, 2007

Some of the photos from the plat_forms 2007 contest are published now! A short description of plat_forms: Software development platforms for web applications (such as Java EE, .NET, Perl, PHP, Python, Ruby on Rails, etc.) are among the most critical factors of development productivity today. The pros and cons of the various platforms are by-and-large known in principle, but how the pros trade off against the cons in one platform and how that compares to another platform is the topic of quasi-religious wars only, not a subject of objective analysis, as almost no data is available that allows such direct comparison.
