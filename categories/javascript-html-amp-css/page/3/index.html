Flickr RSS plugin for jQuery

May 6, 2008

After searching for a suitable Flickr WordPress plugin for ages, I developed my own in less than 40 lines of JavaScript code. It’s free! You may download and use it for your own blog. I recommend to save it as /wp-content/plugins/flickr/flickr.js, if you are using WordPress. As the plugin does not depend on WordPress, you can use it with any other CMS like Drupal too. Using Gdata with jQuery

May 6, 2008

This morning, I had the unbearable desire to find out why I can’t call Google’s AJAX Feed API with jQuery 1.2.3 directly, but using a small proxy script. In theory, it’s just including a script tag which then loads the remote JavaScript and executes a callback function (thus, bypassing the same-origin policy of JavaScript, which would apply for a normal XMLHttpRequest): $.getScript(‘http://adwordsapi.blogspot.com/atom.xml?alt=json-in-script&callback=someCallbackFunction&#8217;); or $.get(‘http://adwordsapi.blogspot.com/atom.xml?alt=json-in-script&#8217;, null, someCallbackFunction, ‘jsonp’);
