Howto: Install PHP 5.5 and PHPUnit on OS X via Homebrew

Here is how to install the latest PHP and PHPUnit versions on OS X using , which became my first choice after testing other package managers. To avoid conflicts, it is generally not recommended to use Homebrew alongside other external OS X package managers such as MacPorts or Fink.
